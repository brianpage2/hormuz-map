# AIS 데이터 소스 비교표

| 후보 | 종류 | 실시간성 | 무료 여부 | 오픈소스 여부 | 호르무즈 bbox 조회 | 장점 | 한계 / 주의점 | 추천도 |
|------|------|---------|-----------|-------------|-----------------|------|--------------|--------|
| **AISStream** | AIS WebSocket API | 높음 | 있음 (베타) | API 자체는 아님, 예제는 공개 | 가능 | bbox 구독 가능, 지도 MVP에 가장 적합, GitHub 예제 존재 | 베타, SLA 없음, throttling 가능, 정책 변동 가능 | **가장 추천** |
| AISHub | AIS REST API | 중간~높음 | 조건부 무료 | 아님 | 가능 | JSON/XML/CSV 지원, 영역 조회 가능 | 무료 사용이 사실상 AIS feed 기여자 중심 | 조건부 추천 |
| Marinesia | 해양 데이터 REST API | 낮음~중간 | 매우 제한적 무료 | 일부 SDK/예제 공개 | 가능 | 문서 깔끔, 빠른 PoC 가능 | 무료 플랜 제한이 너무 큼, 실서비스 지도엔 부적합 | 테스트용 |
| Global Fishing Watch | 공공/분석 API | 낮음 | 있음 | 일부 도구/문서 공개 | 부분 가능 | 공공성 높고 분석용 가치 큼 | 실시간 선박 지도용이 아님, 지연 존재 | 보조 데이터용 |
| VesselFinder API | 상용 AIS API | 높음 | 사실상 없음 | wrapper만 일부 공개 | 가능 | 상용 서비스라 기능 명확 | 무료 오픈소스 후보가 아님 | 상용 대안 |
| MarineTraffic API | 상용 AIS API | 높음 | 거의 없음 | 아님 | 가능 | 업계 인지도 높음 | 상용/제휴 중심, 무료 개발용과 거리가 멂 | 상용 대안 |

---

## 결론

- **1순위**: AISStream — 현재 사용 중, 설정 문제 해결 필요
- **2순위**: AISHub — 수신 장비 없으면 사실상 불가
- **상용 전환 시**: MarineTraffic 또는 VesselFinder

## 현재 상태 (2026-04-14)

- AISStream API 키 발급 완료, WebSocket 연결은 되나 호르무즈 데이터 수신 안 됨
- 원인 미파악 — 무료 티어 지역 제한 or 설정 문제
- → API 키 및 연결 정보는 `api-keys.md` 참고
