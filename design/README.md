# 디자인 파일 폴더

이 폴더에 프로젝트 디자인 관련 파일을 추가하세요.

## 폴더 구조

```
design/
├── wireframes/    ← 화면 구성 와이어프레임 (MD 또는 이미지)
├── content/       ← 콘텐츠 기획 및 SEO 글 초안 (MD 파일)
└── brand/         ← 색상, 폰트, 로고 가이드
```

## 파일 제공 방법

- MD 파일로 작성하여 해당 폴더에 넣으면 Claude가 이를 참고하여 개발합니다.
- 예: `content/hormuz-article.md` — 호르무즈 해협 소개 기사 초안
- 예: `brand/color-guide.md` — 색상 팔레트 및 브랜드 가이드
- 예: `wireframes/homepage.md` — 홈페이지 화면 구성 설명

## 현재 확정된 디자인 방향

- **테마**: 다크 (dark) 모드 기본
- **지도 타일**: CartoDB Dark Matter (어두운 해양 배경)
- **폰트**: Noto Sans KR (한국어 최적화)
- **색상 포인트**: 청록색(teal) 계열 — 해운/항해 느낌

## 뉴스 기사 작성 방법

뉴스 기사는 `../content/news/` 폴더에 MDX 파일로 작성합니다.

```markdown
---
title: "호르무즈 해협 긴장 고조 — 유조선 통행량 변화"
date: "2026-04-13"
description: "최근 중동 정세로 인한 호르무즈 해협 선박 통행량 변화를 분석합니다."
tags: ["호르무즈", "유조선", "중동"]
---

기사 본문을 여기에 작성하세요.
```
